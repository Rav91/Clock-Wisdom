1) Go Flask folder in terminal.
2) Open terminal and type 'pip install -r requirements.txt' or 'pip3 install -r requirements.txt'.
3) Now when all the requirements are installed, type 'app.py' or 'python app.py' or 'python3 app.py' in the same terminal and press enter.
4) The flask server will start working.
5) Now open React in the terminal.
6) Type 'npm start' in the terminal and press enter.
